Keyboard Shortcuts

Actions

o                open file at hunk
s                ignore white space
w                show word diff. word, any charachter, standard
.                go to next hunk
,                go to previous hunk

When Diffing Working Tree

TAB                 switch between staged/unstaged area
h / {super_key}-enter      stage hunk / unstage hunk (in cached mode)
H / {super_key}-backspace  reset hunk
c                   commit
C                   commit, including unstaged
m                   amend previous commit

Other

?                show this help popup
{super_key}-z           undo last action
+                show more context lines
-                show less context lines

Vintageous friendly mode

j                go to next hunk
k                go to previous hunk

Settings

{super_key}-,      Change Settings for current syntax
