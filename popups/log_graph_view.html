Keyboard Shortcuts

Actions

{super_key}-R       Goto symbols, tags, branches
h            Goto HEAD commit
enter        open menu with additional commands
o            open commit in a new view
m            toggle commit details panel on the bottom
a            toggle "--all"
f            edit filters verbatim, e.g. try "--reflog" or "-Ssearch_term"
.            go to next commit
,            go to previous commit
alt+up       go to previous commit
alt+down     go to next commit

Other

?            show this help popup
tab          transition to next dashboard
SHIFT-tab    transition to previous dashboard

Vintageous friendly mode

j            go to next commit
k            go to previous commit

Settings

{super_key}-,      Change Settings for current syntax
